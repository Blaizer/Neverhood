=head1 NAME

Games::Neverhood::DualVar - Overloaded class for dualvars

=head1 DESCRPTION

DualVar is an implementation of L<Scalar::Util::dualvar|Scalar::Util/dualvar NUM, STRING> with an overloaded class, so that the dualvar is stored correctly by L<Storable> and the like.

=head1 METHODS

=head2 new

 my $dualvar = Games::Neverhood::DualVar->new($number, $string);

Returns an object with the value C<$number> when used as a number and C<$string> when used as a string.

=head2 set

 $dualvar->set($number, $string);

Sets the values for the dualvar's number and string.
If undef is specified for either argument, the corresponding value will not be changed.

=head1 OVERLOADED OPERATORS

For number:
 bool 0+
 + - * / % ** << >>
 += -= *= /= %= **= <<= >>=
 < <= > >= == !=
 <=>
 & &= | |= ^ ^=
 neg ! ~
 ++ --
 atan2 cos sin exp abs log sqrt

For string:
 "" qr
 x .
 x= .=
 cmp
 lt le gt ge eq ne

Not overloaded:
 <>
 -X
 ${} @{} %{} &{} *{}
 ~~
 nomethod =

The value for C<fallback> is C<1>
